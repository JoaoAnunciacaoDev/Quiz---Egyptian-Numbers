## 🇺🇸 English

### 📜 About the Project
An interactive educational game designed to test and teach the Ancient Egyptian numeral system. The player must identify the correct hieroglyphs and choose the right answer. The project was built with a focus on a clean interface, smooth transitions (Game Feel), and data-driven architecture.

### ✨ Features
* **Dynamic Quiz System:** Managed through Godot's `Custom Resources`, allowing easy creation and editing of questions without altering the code.
* **Game Feel & Animations:** Immediate visual feedback with code-driven animations (`Tweens`) for correct/wrong answers and screen transitions.
* **Multiple Choice:** Dynamically generated alternatives (A, B, C, D).
* **Cross-platform:** Exported to HTML5/WebAssembly, playable directly in any PC or Mobile browser.

### 🛠️ Technologies Used
* **Engine:** Godot Engine 4.6 (GDScript)
* **Hosting:** GitHub Pages
* **Design/UI:** Godot's native Control Nodes system.

## 🇧🇷 Português

### 📜 Sobre o Projeto
Um jogo educativo e interativo desenvolvido para testar e ensinar o sistema de numeração do Antigo Egito. O jogador precisa identificar os hieróglifos corretos e escolher a alternativa certa. O projeto foi construído focando em uma interface limpa, transições suaves (Game Feel) e arquitetura orientada a dados.

### ✨ Funcionalidades
* **Sistema de Quiz Dinâmico:** Gerenciado através de `Custom Resources` da Godot, permitindo criar e editar perguntas facilmente sem alterar o código.
* **Game Feel & Animações:** Feedback visual imediato com animações criadas via código (`Tweens`) para acertos, erros e transições de tela.
* **Múltipla Escolha:** Alternativas geradas dinamicamente (A, B, C, D).
* **Cross-platform:** Exportado em HTML5/WebAssembly, jogável diretamente em qualquer navegador de PC ou Mobile.

### 🛠️ Tecnologias Utilizadas
* **Engine:** Godot Engine 4.6 (GDScript)
* **Hospedagem:** GitHub Pages
* **Design/UI:** Sistema de Control Nodes nativo da Godot.
